# 📊 E-Commerce Product Funnel Analysis 

## 🚀 Overview

This project dives deep into user behavior across an e-commerce funnel, uncovering critical drop-offs and revenue opportunities. Using Excel, a complete **4-stage customer journey** was mapped — from initial page view to final payment — to understand where and why users leave.

## 🧠 Objective

To analyze user flow, identify bottlenecks, and highlight the **most impactful stage for conversion improvement**.
